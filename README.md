# File Permissions in Liunx

## Objective


The goal of this project was to review file and directory permissions in a Linux environment, determine whether access levels matched the intended authorization, and modify permissions to remove unauthorized access while maintaining appropriate user privileges.

### Skills Learned


- Hands-on experience analyzing file and directory permissions in a Linux environment
- Understanding of Linux permission structures and how to interpret permission strings
- Ability to manage and modify access controls to ensure proper user authorization
- Familiarity with enforcing the principle of least privilege by restricting unnecessary permissions
- Awareness of how access control impacts overall system security

### Tools Used


- Linux command line interface (CLI)
- File permission management tools (chmod)
- File and directory inspection tools (ls -la)
- Directory navigation tools (pwd)
